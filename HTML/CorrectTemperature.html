Instructions for the CorrectTemperature Tab



This tab supports adding new variables that represent corrected temperatures. The processing is different from the “CorrectTemperature.R’ script in that it does not create a new netCDF file but just adds the variables to the working data.frame. For each temperature measurement Atx, the new variables are named ATxF and AtxC. They represent respectively the result of filtering the dynamic-heating correction and of correcting for the time response of the sensor. Common variable names include those where “x” is {‘F1”, “H1”, “H2”} and possibly others.

The “CorrectTemperature.R” script is described at this URL: https://github.com/WilliamCooper/SensibleHeatFlux/blob/master/CorrectTemperature.pdf
